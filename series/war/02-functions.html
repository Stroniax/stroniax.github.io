Learning with a Game of War

PSSharp Series Learning with a Game of War

Another way to think of functions is like mathematical functions. A mathematical function takes input and returns output. For example, the function f(x) = x + 1. This takes input x, and returns output x + 1. PowerShell functions are a little more advanced but the input/transform/output concept transfers.

We're building a card game of war, so the first thing we're going to do is create a card. I'll give top-level explanations of this code but we'll dig into it a little more later on. Run this in your prompt: 

[pscustomobject]@{
    SuitName = 'Spades'
    RankName = 'Ace'
    Value    = 14
}

We see in the output that we just created an object with three properties. This is our representation of a card! Now any time that we want to create a card, we know what code we have to use. But in programming there's a concept called DRY - Don't Repeat Yourself. Since we're planning to use this code more than once, we're going to turn it into a reusable function.

PowerShell has some guidelines to help functions provide a consistent user experience. Commands should be named in the pattern of Verb-Noun. The Noun should be a single, PascalCase noun that describes the object upon which the action (verb) is to be performed. The Verb should be one of the approved PowerShell verbs. To find out which verbs are approved, run Get-Verb. You should see the following output: read through the other verbs to see what you might be able to use in the future. 

Verb        AliasPrefix Group          Description
----        ----------- -----          -----------
Add         a           Common         Adds a resource to a container, or attaches an item to another item
# ... omitted
New         n           Common         Creates a resource
# ... omitted

Since we want to create a card, we'll choose the function name New-Card. Start with the function keyword, followed by the name; then we put the code we want the function to run into a scriptblock by wrapping it in curly brackets { }. Your result will look like the following. 

function New-Card {
    [pscustomobject]@{
        SuitName = 'Spades'
        RankName = 'Ace'
        Value    = 14
    }
}

Now we have a function!
